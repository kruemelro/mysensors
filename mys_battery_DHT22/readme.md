## Sensor with BME280 Temperature Sensor

## Sensor IDs
| ID | Typ |
|:--- |:-------:|
| 1  | Temperature |
| 2  | Humidity |
| 3  | Pressure |
|201 | Sleeptime |
|202 | Force-Reads|
|203 | Config-Outdoor|

## Remote Config Values

| Description       | Sensor-ID          | Payload  |  Default |
| ------------- |:-------------:|:-----:|:-----:
| Sleeptime     | 201 | 1-240 | 2 |
| Force Reads      | 202      |   1-240 | 30 |
| Outdoor | 203      |    0/1 | 0 |
