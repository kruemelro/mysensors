## Sensor with DHT22 Temperature Sensor

## Sensor IDs
| ID | Typ |
|:--- |:-------:|
| 1  | Temperature |
| 2  | Humidity |
|201 | Sleeptime |
|202 | Force-Reads|

## Remote Config Values

| Description       | Sensor-ID          | Payload  |  Default |
| ------------- |:-------------:|:-----:|:-----:
| Sleeptime     | 201 | 1-240 | 5 |
| Force Reads      | 202      |   1-240 | 12 |
